#JS基本知识了解
[TOC]

##JS概述

1.1 JS作用

  • 验证表单（以前的网速较慢，在客户端实现验证更节省时间）
  • 页面特效（PC端的网页效果）
  • 移动端（web 和app）
  • 异步和服务器交互（AJAX）
  • 服务端开服（node.js）

    1.2浏览器的工作原理


    1、User Interface 用户界面，我们所看到的浏览器
    2、Browser engine 浏览器引擎，用来查询和操作渲染引擎
    3、Rendering engine（渲染引擎） 用来显示请求的内容，负责解析HTML、CSS
    4、Networking 网络，负责发送网络请求
    5、JavaScript Interpreter(解析者) Javascript解析器，负责执行JavaScript的代码
    6、UI Backend UI后端，用来绘制雷系组合框和弹出窗口
    7、Data Presistence（持久化）数据持久化，数据存储cookie、HTML5中的seccionStorage

    1.3 弱类型、解释执行脚本语言

    脚本语言：弥补编译语言的不足而存在的，作为补充语言，不需要编译
    解释执行：一行一行解析，解析一行执行一行。

1.4 JS组成

* JS=ECMAScript +DOM +BOM +高级
* ECMAScript：是JavaScript 的语法规范
* DOM(document Object Model)文档对象模型，操作网页上元素的API
* BOM（Browser Object Model）浏览器对象模型，操作浏览器部分功能的API

1.5 输出语句

* console.log("内容")在控制台打印内容
* alert("内容")弹窗显示内容
* document.write("内容")在页面上书写。该语句识别标签

1.6 引入方式

* 内嵌式，将JS代码写在<script></script>中
* 外联式，将Js文件通过外联的方式引入，<script src=""></script>注意：外联的script的标签里不能写任何代码，(实际开发中，多使用该方式引入)

变量

命名规则:

驼峰命名法（从第二个单词起，首字符大写。如：getElementById）

*  1、变量命名必须以字母或者下标""_"或者''$'为开头
*  2、变量名长度不能超过255个字符
*  3、变量名中不允许使用空格 
*  4、不能使用脚本语言中保留的关键字以及保留符号作为变量名。
*  5、变量名区分大小写
*  6、不推荐使用汉语命名

变量使用

定义变量时，尽量不要跨越类型给同一个变量赋值。在强类型语言中，跨越类型定义变量会报错

* 例如： var n1='aaa';n1=99;

交换变量

  • 通过第三方交换，temp;
    • 定义一个中间变量，存放即将被修改的值，
    • 再把中间变量存放的值赋给交换的变量
  • 通过求和的方式，交换变量
    • 先求和，用和减去其中一个，赋值给另一个，最后再用和减去另一个赋值给剩下的一个

      数据类型

      1.1 数据类型划分

      简单数据类型和复杂数据类型
  • 简单数据类型：字符串(string)、数字(Number)、布尔(Boolean)、未定义(undefined)
  • 复杂数据类型：Object、function、Array、Date、RegExp、Error

    字面量：

    固定的值，让你从‘从字面上’表现的值

Number(数字类型)

* 进制：进制包括2进制、8进制（011）、10进制、16进制（0xa）、32进制等....
* 浮点数：容易出现精度丢失问题，所以不判断计算后的两个浮点数是否相等
数值范围（由于内存的限制，ECMAScript 并不能保存世界上所有的数值）
* Number.MIN_VALUE,表示最小值
* 最小值：Number.MIN_VALUE，这个值为： 5e-324
* 最大值：Number.MAX_VALUE，这个值为： 1.7976931348623157e+308
* 无穷大：Infinity（0做除数时，为无穷大）
undefined与null的区别

null和undefined有很大的相似型。null==undefined的结果为true。但是unll===undefined的结果为false。注意：null+数字，结果为数字。但是undefined+数字，结果为NaN
任何数据类型和undefined结果为NaN
任何值和null运算，null可以看做与0运算

NaN (非数值Not a Number的简写)

其类型Number 类型

* console.log("abs"/18)===>>结果是NaN
* undefined与任何数值计算都为NaN;NaN与任何值都不相等，包括他本身
isNaN():任何不能被转换为数值的值购汇导致这个函数返回true;
* isNaN(NaN);// true
* isNaN(“blue”); // true
* isNaN(123); // false

String(字符串类型)

用引号时，可单引号也可双引号，但是不能一单一双。

  • 字符串拼接 ：如果两个变量都是字符串，“+变量+”。用加好拼接，同时加上双引号。
  • 转移字符：无法暑促的字符，先输出/,在输出字符。(“、\、换行等····”)
  • 字符串不可变：开辟空间存储变量时，内存中不会立刻消失，只能二次赋值。

Boolblean（布尔类型）

* Bollean类型有连个字面量：true和false.
* 虽然Boolean 类型的字面值只有两个，但 ECMAScript 中所有类型的值都有与这两个 Boolean 值等价的值 
* true:除0、something、object为true
* false: false、“”、undefined、null、转换时都为false
* if()判断时，会把内容值强行转换成booleab类型

数据类型转换

任何简单类型都可转换成String

* 1、通过+运算，加好充当链接符号，实现隐式转换
* 2、String(变量)，
* 3、变量.toString()  注意：undefined和null不可以转换，即没有toString 方法

任何简单类型转换成Number

* 通过* / - 运算转换成Number，注意，一单别转换的变量中含有非数字字符，容易出现NaN 
*Number(变量)当出现非数字字符串时，会转换成NaN 
* parseInt()和parseFloat(),parseInt转换会保存整数部分，小树部分删除，Float会同时保存小树部分，

任何简单类型转换成Boolean类型

* 第一种：通过构造函数，Boolean(变量)。注意：0代表false，1代表true.
* 0,false,null,undefined,"",代表的是false.其他都为true.
* if()，括号中的内容会转换成Booolean类型
* 第二种：两次取反也可获得，BOOlean 类型。！！11；

操作符的种类：

算数运算符

* 一元运算符：正号、负号、++、--、平方等一个变量就能运算
* 二元运算符：+ - * / 等两个变量才能运算
* 三元运算符：值1？值2：值3

逻辑运算符（|| && !）（或且非）

比较运算符（<、>、==、>=、<=）

赋值运算符（=、+=、-=、*=、/=）

优先级

* 1、（）
* 2、！、-（负数）、++、--、+（正数）、
* 3、*、/、%、
* 4、+、-
* 5、 <、<=、<、>= （一级逻辑运算）
* 6、==、!=、===、!==、 （二级逻辑运算）
* 7、&& （三级级逻辑运算）
* 8、||
* 9、值1？值二：值三（三元运算符）
* 10、 =、+=、-=、*=、/=、%= （赋值运算）

&&和||短路路运算）

* &&链接两个Boolean类型，有一个是false结果就是false.&&寻找false
* ||链接两个boolean类型，有一个是true就是true，找到一个true,就停止寻找
